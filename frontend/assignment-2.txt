Explain with some examples.

Ans:  Datatypes:
    A data type defines the kind of data a variable can hold, like numbers, text, or true/false. 
    Primitive types (e.g., String, Number, Boolean) store single, simple values and are immutable. 
    Non-primitive types (e.g., Objects, Arrays) can hold multiple or complex data and are mutable, allowing changes.

   Types of datatypes:-
   1. Primitive datatypes
    - It can store single values
    - Immutable(cannot be modified or change)
    -Primitive dataypes are:
         1. string
         2. Boolean
         3. number
         4. undefined 
         5. null
         6. big int
         7. symbol
   2.Non-primitive datatypes
    - It can store multiple values
    - Mutable(it can be modified  or change)
    - Non-primitve datatypes are:
         1. class
         2. objects
         3. functions
         4. arrays
         5. math, date, json
